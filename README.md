# Nexus Engine

A real-time collaborative development environment and code execution sandbox built with React, Node.js, and WebSockets.

## Features
- **Real-Time Code Sync:** Live peer collaboration using Socket.io.
- **Secure Code Execution:** Run JavaScript safely in a backend sandbox with latency and memory telemetry.
- **Neural Shell CLI:** Interactive command-line interface inside the app.
- **Cyberpunk UI:** Glassmorphic design with an interactive particle canvas.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Socket.io-client, HTML5 Canvas
- **Backend:** Node.js, Express, Socket.io

## Getting Started

Clone the repo and run both the backend and frontend locally.

```bash
git clone [https://github.com/YOUR_USERNAME/nexus-engine.git](https://github.com/YOUR_USERNAME/nexus-engine.git)
cd nexus-engine

# Run Backend
cd nexus-engine-backend
npm install
npm start

# Run Frontend (in a new terminal)
cd ../nexus-engine-frontend/vite-project
npm install
npm run dev
